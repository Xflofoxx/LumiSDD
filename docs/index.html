LumiSDD

Lightweight Spec Driven Development Framework

v1.0.0 MIT License TypeScript

Install

npm install lumisdk

Features

Spec Management

Define specifications with requirements and constraints

Schema Validation

JSON Schema and OpenAPI support built-in

Code Generation

Generate TypeScript types and documentation

Compliance Tracking

Track requirement compliance over time

Quick Start

import { Spec, TypeScriptGenerator, JsonSchemaValidator } from 'lumisdk';

const spec = Spec.create({
  name: 'UserService',
  version: '1.0.0',
  description: 'User management service',
  schema: {
    type: 'object',
    properties: {
      id: { type: 'string' },
      name: { type: 'string' },
      email: { type: 'string', format: 'email' }
    },
    required: ['id', 'name', 'email']
  },
  requirements: [{
    id: 'req-001',
    description: 'User creation',
    priority: 'high',
    status: 'completed',
    testable: true
  }]
});

const generator = new TypeScriptGenerator();
console.log(generator.generate(spec).code);
